AL-Jouf AI Platform

A regional government needed a bilingual AI platform serving Arabic and English users with real-time LLM-powered Q&A. The challenge: no open-source components supported seamless RTL/LTR switching without layout breaks, and LLM response latency had to stay under 7 seconds while maintaining full conversation context.

Key Contributions

  • Architected a direction-aware component library supporting Arabic RTL + English LTR (i18n) — zero rendering inconsistencies across locales
  • Integrated LLM + RAG chatbot with real-time WebSocket streaming UI achieving sub-7s response; applied SSR/SSG patterns and reusable TypeScript components, cutting future dev time by ~30%
  • Defined React component hierarchy, backend APIs, and Headless Orchard Core CMS integration with best SEO techniques
